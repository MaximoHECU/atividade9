# atividade9

## Descrição
Esta é uma API RESTful desenvolvida em **Node.js** com **Express**, criada como requisito de avaliação para a disciplina de Sistemas Web do curso de Sistemas de Informação na Unileste.

O objetivo principal desta API é para a prátíca acadêmica. A documentação completa dos endpoints foi construída utilizando o **Swagger**.

## 🛠️ Tecnologias Utilizadas
* **[Node.js](https://nodejs.org/):** Ambiente de execução JavaScript.
* **[Express](https://expressjs.com/):** Framework web para Node.js.
* **[Swagger (swagger-ui-express)](https://swagger.io/):** Ferramenta para documentação interativa da API.
* **[Nodemon](https://nodemon.io/):** Ferramenta de desenvolvimento para reiniciar o servidor automaticamente.
* **[Banco de Dados]:** Dados em memória.

## Pré-requisitos
Antes de começar, você vai precisar ter as seguintes ferramentas instaladas em sua máquina:
* [Node.js](https://nodejs.org/en/)
* [Express](https://expressjs.com/)
* [Swagger (swagger-ui-express)](https://swagger.io/)
* [Nodemon](https://nodemon.io/)
